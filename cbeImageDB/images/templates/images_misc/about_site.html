About the CBE Image Database

About

The image database is designed to be a storage location for biofilm images to help with organization of and loss of data. It allows users to upload and search biofilm images. It keeps track of such things as the associated labs, organism, and microscope used.

Experiment

An experiment is all the images that are taken on the same set of samples. Experiments must be named when a user first uploads an image. These can later be searched under the tab “Search Experiments”. {# TODO: Add hyperlink? #} Images can also be uploaded to an existing experiment by selecting it from the search page.

Imagers

An imager is the person who is taking the image. If an imagers name is not on the upload image page, it can be added by selecting create name in the autocomplete box.

Uploading Images

When a user first uploads an image, a new experiment is created under a specific project. An experiment is designed to hold only images relating to the a single experiment. If a user is taking images for multiple experiments, they should use the upload image tab when moving onto a new experiment. A user can upload multiple images to a given experiment. If the user saves the url, they can later upload images to the same experiment. Clicking on an image will show the user more information about a specific image.

Searching

Users can either search by individual attributes or a more general search as well as by the terms that appear in the image descriptions. If nothing is entered in the search, all images with appropriate release dates (TODO: wording?) in the database will be given as a result. Clicking on the image on the search results page will bring the user to a page a details page showing all the images information.

Admin

Admin are able to create new growth mediums, growth substratums, labs, microscope settings, objective mediums, organisms, projects, and vessels. To upload a new image, admin must use the upload image page on the main website. A link to modify image information can be found on the details page for that image.

Admin are able to search all images regardless of their release date. Regular users only have access to images that have a release date equal or less than the current day. Admin are also able to view the associated experiments with images from the image details page.
